# Chat

Um chat simples em tempo real.

##### Tecnologias utilizadas

```Node.js```
```Javascript```
```React```
```Vite```
```Express```
```Socket.io```
```Cors```
```Nodemon```


##### Como utilizar?

> **Clone o repositório**

```bash
$ git clone https://github.com/duutheboy/react-realtime-chat.git
```
> **Acesse os diretórios**

```bash
$ cd server
$ cd client
```
> **Rode os 2 diretórios**
```bash
$ npm run dev
```
> **Acesse em 2 navegadores, ou abas, diferentes**
```bash
$ http://localhost:5173/
```
